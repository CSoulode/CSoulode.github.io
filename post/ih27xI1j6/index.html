蓝桥杯 BEGIN-2 入门训练 序列求和

2020-03-10 1 min read # 蓝桥杯(入门训练)

问题描述
求1+2+3+...+n的值。

输入格式
输入包括一个整数n。

输出格式
输出一行，包括一个整数，表示1+2+3+...+n的值。

样例输入
4
样例输出
10

样例输入
100
样例输出
5050

数据规模与约定
1 <= n <= 1,000,000,000。

#include <bits/stdc++.h>
using namespace std;

unsigned long long n, a, i = 1;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(0);
    cin >> n;
    for(i; i <= n; i++)
        a += i;
    cout << a << endl;
    
    return 0;
}
